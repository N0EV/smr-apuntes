# Apuntes SMR - Ciclo Completo (CyL - Curriculum LOE)

Este repositorio contiene mis apuntes, comandos, laboratorios y scripts de Sistemas Microinformáticos y Redes (SMR), adaptados al curriculo oficial de la Comunidad de Castilla y León (normativa LOE clásica).

El objetivo es mantener un espacio centralizado, limpio y accesible para estudiar, repasar y compartir conocimiento técnico de todo el ciclo.

---

Haz clic en cada modulo para acceder a su carpeta correspondiente. Cada seccion incluye sus propios requisitos de software, teoria y practicas.

## Como Colaborar

Este es un proyecto abierto a la comunidad. Si eres estudiante de SMR (especialmente en Castilla y León) y quieres mejorar el temario, corregir errores o añadir tus practicas, puedes hacerlo siguiendo estos pasos:

1. Haz un Fork de este repositorio.
2. Crea una rama con tu mejora: git checkout -b feature/nueva-mejora
3. Haz tus cambios y haz un commit: git commit -m 'Actualiza apuntes de HTML/CSS en AW'
4. Sube la rama: git push origin feature/nueva-mejora
5. Abre un Pull Request para que pueda revisar y fusionar tus cambios.
